CardMachingTask

===

This task is given my Alantrix Geim Studios as a recruiting process

Card match contain 4 script

1. AudioHandler.cs -> Singleton audio script manage all sound related functionality.
2. Card.cs -> Handle card prefab flip, rotate and reset rotate functionality.

3\) CardManager.cs -> It is a singleton class who handles card.cs, score system and manage card selection functionality.

4\) LevelState.cs -> It is a public class work and level identifier and decide the grid layout.
